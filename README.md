1. Tema
O trabalho final tem como tema o desenvolvimento de um sistema de auto presença para o aluno da UFOP integrado ao aplicativo moodleUFOP

2. Escopo
Este projeto terá as seguintes funcionalidades:
Criar turmas para uma determinada disciplina
Adicionar/criar alunos para turmas assim como professores responsáveis pela disciplina
Alterar/modificar informações sobre a presença do discente/docente
Visualizar informações em gráficos sobre a presença
Além de funcionalidades que utilizam da presença do aluno para outras questões (Programas de auxílio universitário) 
Excluir dados sobre usuários 
Usuário administrador 
Os usuários podem administrar diferentes páginas com informações diferentes
